# SlotMachine

- Верстка сделана под едигственное разрешение из фигмы - 800 * 360. Хедер и футер резиновые, контент фиксированный,
  так как нет других вариантов верстки в фигме.
- Для написания логики выбрана первая игра - Royal Coins. Логика работы согласно ТЗ.
- Победа - 3 одинаковых элемента в ряд. Код пердусматривает расширение на другие победные кейсы.
- В localStorage хранятся стейты выбранных страниц, монет и имеющихся и максимальных звезд.
- this.chanceRange - это то, что влияет на шанс победы. Для быстрого теста победного кейса изените значение на 9.
